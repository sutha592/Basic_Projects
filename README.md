# Basic Projects

A collection of projects developed to explore various concepts in Machine Learning, Computer Vision, Data Analysis, Logic Programming, and Game Development. These projects demonstrate practical implementation of different technologies and problem-solving approaches.

## Projects Included

### 1. Stock Forecast App

* Predicts stock prices using time-series forecasting techniques.
* Visualizes historical trends and future predictions.
* Built using Python and data analysis tools.

### 2. Object Detection and Tracking

* Real-time object detection and tracking using YOLOv8.
* Demonstrates Computer Vision and Deep Learning concepts.
* Capable of identifying and tracking objects in video streams.

### 3. Tic Tac Toe Game (Prolog)

* Classic Tic Tac Toe game implemented using Prolog.
* Demonstrates logic programming, facts, rules, and decision-making.
* Showcases the use of AI concepts in game logic.

### 4. Snake Game

* Interactive Snake Game developed using Python.
* Features score tracking and keyboard controls.
* Demonstrates fundamental game development concepts.

## Technologies Used

* Python
* Prolog
* Machine Learning
* Computer Vision
* YOLOv8
* Data Analysis
* Streamlit

## Purpose

This repository serves as a collection of learning and practice projects that strengthen programming, logical thinking, and software development skills through hands-on implementation.
